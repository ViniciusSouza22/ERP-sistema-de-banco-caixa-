Observações do projeto:

Este sistema foi projetado para ser escalável, permitindo a implementação de novas funcionalidades. Sugere-se a integração de APIs de pagamento (como Pix ou Boleto) para expandir as capacidades do projeto.

O código está aberto para que desenvolvedores iniciantes possam explorar e aplicar novas ideias. Para uma segurança mais robusta nas APIs, recomenda-se o uso de linguagens de alto desempenho e tipagem forte, como Java, C# ou C++.
